
# SubsFlow

A high-performance, minimalist subscription manager inspired by the aesthetics of Claude AI and Linear.

## Features

- **Trigram Search**: Instantly find subscription groups using advanced fuzzy matching.
- **Real-time Analytics**: Monthly and yearly cost calculations update at 120fps (visually).
- **Custom UI**: No default browser inputs. Fully styled, custom dropdowns and text fields.
- **Dual Theme**: Classic Dark (default) and Light mode.
- **Localization**: Full English and Russian support.
- **Insights Widget**: Fading inspirational quotes that update every 30 seconds.
- **Local Storage**: Data persists automatically in your browser.

## Tech Stack

- React 18+
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
- Framer Motion principles (implemented via CSS transitions for performance)

## Getting Started

1. Clone the repository.
2. Run `npm install`.
3. Run `npm start`.

## Usage

1. Create a "Flow" (a category, e.g., "Work", "Entertainment").
2. Add subscriptions with price and billing cycle.
3. Watch the totals calculate instantly.
4. Hover over the right side or rows to reveal actions and widgets.
